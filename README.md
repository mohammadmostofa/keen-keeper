
keep keeper project

A simple React + Vite based contact interaction tracker website

---

 Features

- 📞 Add Call interactions
- 💬 Add Text interactions
- 🎥 Add Video interactions
- 📊 Pie Chart analytics (Recharts)
- 🔍 Filter timeline by action type
- ⚡ Duplicate prevention (no repeated entries)
- 🌐 Fully responsive UI
- ☁️ Deployed on Vercel/Netlify

---

Tech stack

- React.js
- Vite
- Context API
- React Router
- Recharts
- React Toastify
- Tailwind CSS (if used)
